⚠ Ouverture depuis file:// détectée

MSAL ne fonctionne pas en protocole file://. Lance un mini-serveur local :

Option 1 — Python (le plus simple) :
Ouvrir un terminal dans le dossier du fichier, taper :
python -m http.server 8080
Puis ouvrir http://localhost:8080/cockpit_m365.html

Option 2 — VS Code :
Installer l'extension "Live Server", clic droit sur le fichier → "Open with Live Server"
